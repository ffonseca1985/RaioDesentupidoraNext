## 🚀 Como Executar

### Pré-requisitos
```bash
Node.js 18+ 
npm ou yarn
```

### Instalação
```bash
# Clone o repositório
git clone [url-do-repositorio]

# Entre no diretório
cd RaioDesentupidoraNext

# Instale as dependências
npm install

# Execute em modo de desenvolvimento
npm run dev

# Acesse http://localhost:3000
```

### Build para Produção
```bash
# Gerar build otimizado
npm run build

# Executar em produção
npm start
```
